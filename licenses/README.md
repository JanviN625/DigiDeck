# Third-Party Licenses

This directory is dedicated to storing the license agreements for any third-party APIs, libraries, or packages used in the DigiDeck application that require their license to be distributed alongside the software.

Currently, the main project is licensed under the AGPLv3 (see the `LICENSE` file in the root directory), which also covers our use of `essentia.js`. If you add any other libraries with different licensing requirements (like MIT, Apache 2.0, etc.), please place a copy of their license files here for full compliance!
